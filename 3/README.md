# Киноглушитель 

Штука, которая принимает на вход видео, саундтрек и субтитры из Интернета, и отдаёт чёрно-белое видео с субтитрами из Интернета и саундтреком из Интернета. 

## Install

~ git  clone

~ npm install

~ npm start

### Плюсы
